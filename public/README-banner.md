# qiqirn-banner.png

The sidebar title uses `public/qiqirn-banner.png` as its background art
(see `src/components/layout/Sidebar.tsx`).

To set it: save the qiqirn merchant artwork as `public/qiqirn-banner.png`.
A wide crop (roughly 2:1 or wider) works best — the title sits over its
left third, which is darkened by a gradient overlay for legibility.

Until the file is added, the title simply renders over the normal card
background (no broken image is shown).
